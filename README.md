
Frontend Setup (React)
# Prerequisites  
 - Node.js v24.1.0 (or compatible)

# Clone the repo 
 - git clone https://github.com/charith-lansakara/book-rental-platform-fe.git

# Go to the project directory
 - cd book-rental-platform-fe

# Install dependencies:
 - npm install

# Start the React app:
 - npm run dev

React frontend will run at http://localhost:5173
